## 📚 Penelitian Utama

### **RESEARCH** - Model CART untuk Klasifikasi Tingkat Pengangguran Terbuka

Repository ini memuat seluruh kode program dari penelitian saya mengenai:

**Judul**: *Pengklasifikasian Tingkat Pengangguran Terbuka di Provinsi Jawa Tengah Menggunakan Model CART*

**Deskripsi**:
- 🎯 **Tujuan**: Mengklasifikasi tingkat pengangguran terbuka di Jawa Tengah menggunakan algoritma Classification and Regression Trees (CART)
- 📈 **Data**: Memanfaatkan data sosial ekonomi dari berbagai sumber
- 🔍 **Metodologi**: Implementasi CART untuk decision tree classification
- 📍 **Wilayah Studi**: Provinsi Jawa Tengah, Indonesia

**Repository**: https://github.com/Dana69h/RESEARCH
